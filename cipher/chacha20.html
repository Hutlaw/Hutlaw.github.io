ChaCha20 Explained

What is ChaCha20? ChaCha20 is a stream cipher that generates a pseudorandom keystream which is XORed with plaintext to produce ciphertext.

Key: A 256-bit (64 hex digit) key used to initialize the cipher.

Nonce: A 96-bit (24 hex digit) number that should be unique for each encryption with the same key.

Counter: A 32-bit number (often starting at 1) that increments for each 64-byte block.
